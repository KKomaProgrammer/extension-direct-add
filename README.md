# Extension Direct Add Helper

Chrome MV3 helper project.

## Files

- manifest.json
- background.js
- content-script.js
- content-style.css
- popup.html
- popup.css
- popup.js

## Notes

The project detects ZIP and GitHub repository links on pages and starts a normal Chrome download after the user presses the download button. Chrome still requires the user to load unpacked extensions manually from the browser extension management page.
